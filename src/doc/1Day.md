#### 태그 선택자
```css
tag {
    property : value;
}
```
#### id 선택자
```css
#id {
    property : value;
}
```
#### class 선택자
- 어려개의 선택이 가능하다
```css
.class {
    property : value;
}
```
#### 자손 선택자
```css
.parent .child {
    property : value;
}
```
#### 다중 선택자
```css
.class#id {
    property : value;
}
```
